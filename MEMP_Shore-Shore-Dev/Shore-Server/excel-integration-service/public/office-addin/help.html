MEMP_Shore Report Uploader - Help Guide

Single-Click Upload (Excel Add-in)

Step 1: Download the vessel report template from the MEMP_Shore application.
Step 2: Open the template in Microsoft Excel (Excel 2016 or later).
Step 3: Fill in your vessel daily report data in the VesselDailyReports sheet.
Step 4: Click the "Send Report" button in the Home ribbon.
Step 5: Enter your API URL and authentication token, then click Validate and Send.

Authentication

To upload reports, you need:

  • API URL: The base URL of your MEMP_Shore API (e.g., http://localhost:4000)
  • Auth Token: Your JWT authentication token from the MEMP_Shore web application

How to Get Your Auth Token

1. Log in to the MEMP_Shore web application
2. Open Browser Developer Tools (F12)
3. Go to Application â†’ Local Storage
4. Find "token" or "authToken"
5. Copy the value and paste it into the add-in

Troubleshooting

Issue: "VesselDailyReports sheet not found"

Solution: Make sure you're using the official MEMP_Shore template and haven't renamed the sheets.

Issue: "Upload failed with status 401"

Solution: Your authentication token has expired. Get a new token from the web application.
